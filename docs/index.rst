**(\*) = same-user (user who ran the command)**

**(#) = needs hex code (Eg #db7991 or just db7991**

**(%) = user input (text)**

**($) = user input (attachment)**

**(@) = user mention**



**Profile commands**
-----
`#=profile create`

`#=profile addtext (%)`

`#=profile view (*)`

`#=profile background (#)`

`#=profile color (#)`

**Posting Commands**
-----
#=post create <ARGS>

  Arguments:
   `-noavatar` | will not show avatar next to name

   `-text(%)`  | text of post
   
   `-title(%)`  | title of post
 
   Note: the `-` and `()` are required.
